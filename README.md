## GitHub Finder

An app that connect to the Github API, and list all public repositories from an user, and the last commits for the repositories.

## Project stack

- Framework: React
- Styleguide: StoryBook
- State manager: Redux
- Styles: CSS Modules
- Tests: Jest
- Router: React router
